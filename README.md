# Restaurant Ordering System

A responsive ReactJS-based restaurant ordering system with menu display, cart functionality, and order tracking. Built for CSCI426 Advanced Web Programming.

## Features
- Browse restaurant menu items with images and descriptions
- Add items to cart and view total
- Track your orders dynamically
- Responsive design for desktop and mobile
